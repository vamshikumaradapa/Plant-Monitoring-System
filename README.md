# 🌱 Plant Monitoring System

## 📌 Overview
An IoT-based plant monitoring system that tracks soil moisture and environmental conditions in real-time.  
It helps maintain plant health by providing continuous data and insights.

---

## 🚀 Features
- Soil moisture monitoring  
- Temperature & humidity tracking  
- Real-time data display  
- Simple and easy-to-use system  
- Expandable for automation  

---

## 🛠️ Tech Stack
- Arduino / ESP32  
- Embedded C  
- Serial Monitor (for output)  

---

## 🔧 Components Required
- Arduino / ESP32  
- Soil Moisture Sensor  
- DHT11 / DHT22 Sensor  
- Jumper Wires  
- Power Supply  

---

## ⚙️ Working Principle
- Soil moisture sensor reads water level in soil  
- DHT sensor measures temperature & humidity  
- Microcontroller processes the data  
- Output is displayed on Serial Monitor  

---

## 🧰 Setup Instructions
1. Connect all components properly  
2. Upload `main.ino` to Arduino/ESP32  
3. Open Serial Monitor  
4. Observe real-time sensor readings
